# 🚀 Advanced Developer Portfolio Blog

Este es un proyecto de blog interactivo desarrollado para demostrar habilidades avanzadas en inglés técnico, pensamiento crítico y desarrollo web frontend (HTML5, CSS3, JS). El diseño sigue una estética **Futurista/Cyberpunk** inspirada en interfaces de sistemas operativos avanzados.

## 📋 Características
* **Diseño Futurista:** Interfaz de "Glassmorphism" con efectos de neón y escaneo láser.
* **Filtro Interactivo:** Sistema de navegación dinámico con JavaScript para filtrar posts.
* **Contenido Académico:** 4 publicaciones desarrolladas con estructuras gramaticales complejas (Voz pasiva, condicionales, tiempos perfectos).
* **Responsive Design:** Adaptable a diferentes tamaños de pantalla.

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructura semántica.
* **CSS3:** Animaciones personalizadas, Grid y Flexbox.
* **JavaScript:** Lógica de filtrado y transiciones de estado.
